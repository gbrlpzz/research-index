This repository serves as the central index for your open research.

## Structure

- **00. catalogue/**: Contains the lists of all your research artifacts (papers, datasets, apps, etc.).
- **01. refs/**: Contains the shared bibliography (`references.bib`) used across all your papers.
- **99. templates/**: Contains templates for creating new artifacts.

## Usage

This index is automatically managed by the Open Research Interface. You generally don't need to edit files here manually, except for the shared bibliography if you wish to add references externally. repos
